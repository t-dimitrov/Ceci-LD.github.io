About Me

Hi! I'm Tsvetelin, a game programmer and software engineering student from Bulgaria, and I've been making games since 13 and playing games from an even younger age. I published my first mobile game on Google Play in August, 2019. Currently I am in the last semester of my Software Engineering Bachelor program at VIA University College in Denmark.

Here you can see some of the projects I have had the opportunity to contribute to, and some personal projects that I have created myself.

Personal Information

Since I was 9 years old I have never lived in a single country more than 2 consecutive years. Visualized Sorting Algorithms

  • Project Status: Complete
  • Project Type: Personal
  • Project Duration: ~2 days
  • Software Used: VS Code
  • Languages/Libraries Used: JavaScript, p5.js

I have studied algorithms and data structures in my 3rd semester, but as time passes knowledge seems to get forgotten. Instead of creating such algorithms to use in individual projects, programmers tend to use pre-existing libraries with these algorithms already implemented. When I started learning JavaScript, I wanted to make sure I haven't forgotten about these algorithms and applied them in a small project.

I have implemented three sorting algorithms: Bubble sort, heap sort, and quick sort. The application starts with one unsorted array and depending on the algorithm you choose to use, the array will begin to visually get sorted.
